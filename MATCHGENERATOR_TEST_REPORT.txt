Issue Found: Column Mismatch in Bulk Insert
──────────────────────────────────────────────

Problem:
  ❌ buildRoundRobinMatchRows() missing columns: bracket_match_id, 
     next_bracket_match_id
  ❌ buildBracketMatchesFromArray() included these columns
  ❌ Laravel bulk insert() expects consistent column count across rows
  ❌ Result: SQLSTATE[21S01] "Column count doesn't match value count"

Root Cause:
  Two different methods generating match rows with different column structures
  Bulk insert attempted on mixed array structure

Solution Applied:
  ✅ Added missing columns to buildRoundRobinMatchRows()
  ✅ Set bracket_match_id = null for league/round-robin matches
  ✅ Set next_bracket_match_id = null for league/round-robin matches
  ✅ Ensured all match arrays have identical column structure

Result:
  ✅ Bulk insert now succeeds
  ✅ All matches properly saved to database
